# Быстрый старт

## Что было создано

Полнофункциональная платформа для создания Telegram-ботов интернет-магазинов со следующими компонентами:

### ✅ Backend API
- REST API на Node.js + Express + TypeScript
- Prisma ORM с полной схемой базы данных
- JWT авторизация
- Все основные эндпоинты согласно ТЗ:
  - Авторизация и регистрация
  - Управление ботами
  - Категории товаров (древовидная структура)
  - Товары с множественными изображениями
  - Заказы и статусы
  - Клиенты (CRM)
  - Корзина
  - Настройки (бренд, платежи, уведомления, меню, шаблоны)

### ✅ Frontend (Админ-панель)
- Next.js 14 с App Router
- React 18 + TypeScript
- Ant Design для UI
- Страницы:
  - Главная
  - Вход / Регистрация
  - Панель управления (Dashboard)
- API клиент с автоматической авторизацией

### ✅ Telegram Bot
- Модуль на node-telegram-bot-api
- Базовая логика для:
  - Приветствие
  - Каталог товаров
  - Корзина
  - Заказы
  - Поддержка
- Интеграция с Backend API

### ✅ База данных
- Полная схема Prisma с 20+ таблицами
- Поддержка всех функций из ТЗ
- Миграции готовы к применению

## Следующие шаги

1. **Установите зависимости:**
   ```bash
   npm run install:all
   ```

2. **Настройте базу данных:**
   - Создайте PostgreSQL базу
   - Настройте `backend/.env`
   - Примените миграции: `cd backend && npx prisma migrate dev`

3. **Запустите проект:**
   ```bash
   npm run dev
   ```

4. **Откройте в браузере:**
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:3001

5. **Зарегистрируйтесь и начните работу!**
